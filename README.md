## 🧭 Модель v.0.2: Переменные-сенсоры

Это не «оценка личности». Это **инструмент для моментальной диагностики состояния**.

| Переменная | Вопрос для калибровки |
|------------|----------------------|
| **`g`** (Gravity) | Насколько это лично? Где моё «Я» в этом? |
| **`u`** (Utility) | Что это даст? Как применить? |
| **`ρ`** (Rho) | Где отзывается в теле? Насколько это «густо»? |
| **`f`** (Freedom) | Насколько я свободен в выборе сейчас? |
| **`S`** (Sovereignty) | Могу ли я сказать «нет»? Кто выбирает? |
| **`E`** (Energy) | Есть ли ресурс на действие? |
| **`P`** (Pressure) | Насколько среда требует реакции? |
| **`Δ`** (Delta) | Кто инициирует сдвиг: я или внешний шум? |

**Формула ценности (черновик):**
V = (g × u × ρ) × (S × f × Δ_self) / P

> Чем выше `V` — тем больше потенциал для интеграции и роста `E_понимание`.
